## Как использовать

### Шаг 1: Откройте конструктор
Нажмите кнопку "⚙️ Конструктор варианта 2" в результатах расчета

### Шаг 2: Нажмите кнопку очистки
В конструкторе внизу слева найдите красную кнопку:
```
[🗑️ Очистить предыдущие]  [Сбросить]  [Применить к расчету]
```

### Шаг 3: Подтвердите действие
Появится диалог:
```
Очистить все предыдущие результаты расчетов?

Останется только последний расчет.

[Отмена]  [ОК]
```

### Шаг 4: Результат
- Все старые результаты удалены
- Остался только последний расчет
- Конструктор автоматически закрыт
- Показано сообщение: "✓ Предыдущие результаты очищены"

## Что делает функция

### На обычной странице
```javascript
// Очищает div с результатами, оставляя только последний
const resultDiv = document.getElementById('result');
const lastResult = resultDiv.lastElementChild;
resultDiv.innerHTML = '';
resultDiv.appendChild(lastResult);
```

### В расширении
```javascript
// Удаляет все элементы истории кроме последнего
const historyItems = historyContainer.querySelectorAll('.history-item');
for (let i = 0; i < historyItems.length - 1; i++) {
  historyItems[i].remove();
}
```

### Дополнительно
```javascript
// Закрывает конструктор
window.closeVariant2Constructor();

// Показывает уведомление
alert('✓ Предыдущие результаты очищены');
```

## Когда использовать

### ✅ Рекомендуется использовать:
1. После нескольких расчетов подряд (Сбербанк → ВТБ → Дом.РФ)
2. Когда на странице накопилось много результатов
3. Перед важным расчетом, чтобы избежать путаницы
4. Если случайно открыли конструктор для старого результата

### ⚠️ Проблема которую решает:
Когда вы делаете несколько расчетов подряд, на странице накапливаются ВСЕ результаты. У каждого результата есть своя кнопка "Конструктор варианта 2". Если случайно нажать на кнопку от СТАРОГО результата, открывается конструктор со старыми данными и настройками (например, скидка 50% от Сбербанка применяется к ВТБ).

## Workflow

### Правильный workflow:
1. Рассчитали Сбербанк → открыли конструктор → настроили скидку 50%
2. Рассчитали ВТБ
3. **Открыли конструктор → нажали "🗑️ Очистить предыдущие"**
4. Теперь на странице только ВТБ (результат Сбербанка удален)
5. Работаете с ВТБ без риска перепутать с предыдущими расчетами

### Альтернативный workflow (без очистки):
1. Рассчитали Сбербанк
2. Рассчитали ВТБ
3. Рассчитали Дом.РФ
4. На странице ВСЕ три результата
5. **⚠️ ВНИМАТЕЛЬНО смотрите какую кнопку "Конструктор" нажимаете!**
6. Легко перепутать и открыть конструктор для старого результата
