# Исправления расширения - Финальная версия

## Исправленные проблемы

### 1. Копирование - исправлено ✅
**Проблема**: Копировалось всё (Банк, Остаток долга, Вариант 1, Вариант 2) вместо только Варианта 1

**Решение**: 
- Изменена логика в `extension/calculator-main.js`
- Теперь используется точное совпадение `line === 'Вариант 1:'` вместо `startsWith()`
- Добавлена проверка на пустые строки перед захватом
- Копируется только содержимое Варианта 1 БЕЗ заголовка "Вариант 1:"

**Тест**: ВТБ 12.11.2021, ОСЗ 3 342 490, квартира
**Ожидаемый результат**: `Имущество 3676,74\nИТОГО тариф/ взнос 3676,74`

### 2. Конструктор показывает неправильные риски - исправлено ✅
**Проблема**: Для домов показывал "Моя квартира" вместо "Бастион"

**Решение**:
- Добавлена проверка типа объекта в `openVariant2Constructor()`
- Определяется `isHouse` на основе `objectType` (house_brick, house_wood, house)
- Заголовки рисков динамически меняются: "Бастион" для домов, "Моя квартира" для квартир
- Исправлено в обоих файлах: `calculator-variant2-ui.js` и `extension/calculator-variant2-ui.js`

**Тест**: Дом РФ от 23.07.2025, ОСЗ 3648641,81, дом дерево/дерево 2025
**Ожидаемый результат**: Конструктор показывает "Бастион: отделка и инженерное оборудование"

### 3. Версия расширения обновлена
- Версия изменена с `1.0.0` на `1.0.1` в `manifest.json`
- Это заставит браузер обновить расширение

## Инструкции по обновлению

1. **Удалите старое расширение**:
   - Откройте `edge://extensions/` или `chrome://extensions/`
   - Найдите "Ingosstrakh Calculator"
   - Нажмите "Удалить"

2. **Установите новую версию**:
   - Включите "Режим разработчика" (Developer mode)
   - Нажмите "Загрузить распакованное расширение" (Load unpacked)
   - Выберите папку `extension`

3. **Проверьте версию**:
   - В списке расширений должна быть версия `1.0.1`
   - Иконка должна быть цветной (не серой)
