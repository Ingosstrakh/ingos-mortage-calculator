# 🔧 Переустановка расширения

## Проблема: Серая иконка (расширение не загружается)

Это происходит из-за ошибки при обновлении. Нужно переустановить расширение.

---

## 📋 Инструкция по переустановке:

### Шаг 1: Удалите старое расширение
1. Откройте `chrome://extensions/`
2. Найдите "Ингосстрах - Калькулятор страхования"
3. Нажмите кнопку **"Удалить"**
4. Подтвердите удаление

### Шаг 2: Перезапустите браузер (рекомендуется)
1. Закройте Chrome полностью
2. Откройте Chrome заново

### Шаг 3: Установите расширение заново
1. Откройте `chrome://extensions/`
2. Включите **"Режим разработчика"** (переключатель в правом верхнем углу)
3. Нажмите **"Загрузить распакованное расширение"**
4. Выберите папку `extension` в вашем проекте
5. Расширение должно загрузиться с цветной иконкой ✅

### Шаг 4: Проверьте работу
1. Кликните на иконку расширения
2. Должна открыться боковая панель
3. Введите тест-кейс и нажмите "Рассчитать"

---

## ⚠️ Если всё ещё не работает:

### Проверьте ошибки:
1. На странице `chrome://extensions/`
2. Найдите расширение
3. Нажмите **"Ошибки"** (если есть красная кнопка)
4. Посмотрите текст ошибки

### Частые проблемы:

**1. "Service worker registration failed"**
- Решение: Удалите расширение, перезапустите Chrome, установите заново

**2. "Could not load icon"**
- Решение: Убедитесь, что в папке `extension/icons/` есть все 4 иконки:
  - icon16.png
  - icon32.png
  - icon48.png
  - icon128.png

**3. "Manifest file is missing or unreadable"**
- Решение: Проверьте, что файл `extension/manifest.json` существует и читается

**4. "Invalid manifest"**
- Решение: Я обновил manifest.json с английскими названиями (избежание проблем с кодировкой)

## 🎯 Альтернативный способ (если не помогло):

### Создайте новую папку:
1. Создайте папку `extension-clean`
2. Скопируйте туда ВСЕ файлы из `extension/`
3. Установите расширение из `extension-clean/`

### Команда PowerShell:
```powershell
# Создать чистую копию
New-Item -ItemType Directory -Path "extension-clean" -Force
Copy-Item -Path "extension/*" -Destination "extension-clean/" -Recurse -Force
```

## ✅ После успешной установки:

### Проверьте:
1. ✅ Иконка цветная (не серая)
2. ✅ Клик по иконке открывает боковую панель
3. ✅ Расчет работает
4. ✅ Конструктор варианта 2 открывается
5. ✅ Горячие клавиши работают (`Alt+Shift+C`)

### Настройте горячие клавиши:
1. Откройте `chrome://extensions/shortcuts`
2. Найдите "Ingosstrakh Calculator"
3. Убедитесь, что установлены:
   - `Alt+Shift+C` - Calculate from selected text
   - `Alt+Shift+I` - Open calculator panel

## 📝 Что изменилось в manifest.json:

**Было (русский):**
```json
{
  "name": "Ингосстрах - Калькулятор страхования",
  "description": "Автоматический расчет ипотечного страхования для Max и Outlook",
  "default_title": "Открыть калькулятор"
}
```

**Стало (английский):**
```json
{
  "name": "Ingosstrakh Calculator",
  "description": "Insurance calculator for Max and Outlook",
  "default_title": "Open calculator"
}
```

Это сделано для избежания проблем с кодировкой UTF-8 в Chrome.

5. Пришлите мне текст ошибки
