## ✅ Исправление 1: Копирование только варианта 1

### Проблема:
Копировалось ВСЁ:
```
Банк: ВТБ
Остаток долга: 3342490,00 ₽
Надбавка 10%: 334249,00 ₽
Страховая сумма: 3676739,00 ₽
Вариант 1:
Имущество 3676,74
ИТОГО тариф/ взнос 3676,74
Вариант 2 (повышенные скидки + доп. риски):
...
```

### Решение:
Теперь копируется ТОЛЬКО вариант 1 БЕЗ заголовка:
```
Имущество 3676,74
ИТОГО тариф/ взнос 3676,74
```

### Изменения в коде:
```javascript
// Было: line.includes('Вариант 1:')
// Стало: line.startsWith('Вариант 1:')

// Добавлена проверка foundVariant1
let foundVariant1 = false;

if (line.startsWith('Вариант 1:')) {
  foundVariant1 = true;
  captureVariant1 = true;
  continue; // Пропускаем заголовок
}

if (foundVariant1 && captureVariant1) {
  // Захватываем только после "Вариант 1:"
  if (line.startsWith('Вариант 2') || line.startsWith('Вариант 3')) {
    break; // Останавливаемся на следующем варианте
  }
  
  if (line) {
    textForClipboard += line + '\n';
    
    if (line.includes('ИТОГО')) {
      break; // Останавливаемся после ИТОГО
    }
  }
}
```

**Файл:** `calculator-main.js`

## ✅ Исправление 2: Конструктор варианта 2 в расширении

### Проблема:
Кнопка "Конструктор варианта 2" не работала в sidepanel из-за Content Security Policy (CSP). Атрибут `onclick` в HTML не выполняется в расширениях.

### Решение:
Добавлен обработчик события после вставки HTML:

```javascript
function displayResult(result) {
  resultsContent.innerHTML = result;
  resultsSection.style.display = 'block';
  resultsSection.scrollIntoView({ behavior: 'smooth', block: 'nearest' });
  
  // Добавляем обработчик для кнопки конструктора
  const constructorBtn = resultsContent.querySelector('button[onclick*="openVariant2Constructor"]');
  if (constructorBtn) {
    constructorBtn.removeAttribute('onclick'); // Удаляем onclick
    constructorBtn.addEventListener('click', () => {
      if (typeof window.openVariant2Constructor === 'function') {
        window.openVariant2Constructor();
      } else {
        showToast('❌ Конструктор варианта 2 недоступен', 'error');
      }
    });
  }
}
```

**Файл:** `extension/sidepanel.js`

## 📝 Инструкция по обновлению:

1. Откройте `chrome://extensions/`
2. Найдите "Ингосстрах - Калькулятор страхования"
3. Нажмите кнопку "Обновить" (🔄)
