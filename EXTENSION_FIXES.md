# 🔧 Исправления расширения

## Дата: 5 марта 2026

### ✅ Исправлено:

## 1. Копирование только варианта 1 (без заголовка)

**Проблема:** Копировались все варианты с заголовками  
**Решение:** Теперь копируется только вариант 1 БЕЗ фразы "Вариант 1:"

**Что копируется:**
```
имущество 5 000,00
жизнь заемщик 3 500,00
титул 2 000,00
ИТОГО тариф/ взнос 10 500,00
```

**Файл:** `calculator-main.js`

## 2. Конструктор варианта 2 в расширении

**Проблема:** Модальное окно конструктора не отображалось в sidepanel  
**Решение:** 
- Добавлен контейнер `variant2-constructor-modal-container` в `sidepanel.html`
- Функция `ensureVariant2ConstructorModal()` теперь ищет контейнер или использует body

**Файлы:** 
- `extension/sidepanel.html` - добавлен контейнер
- `calculator-variant2-ui.js` - исправлено добавление модального окна

## 3. Горячие клавиши

**Проблема:** Горячие клавиши не работали (конфликт с DevTools)  
**Решение:** Изменены комбинации клавиш

**Было:**
- `Ctrl+Shift+C` - Рассчитать (конфликт с DevTools)
- `Ctrl+Shift+I` - Открыть панель (конфликт с DevTools)

**Стало:**
- `Alt+Shift+C` - Рассчитать ✅
- `Alt+Shift+I` - Открыть панель ✅

**Файлы:**
- `extension/manifest.json` - обновлены команды
- `extension/README.md` - обновлена документация
- `extension/sidepanel.html` - обновлены подсказки

## 4. Инициализация базы варианта 2

**Проблема:** База пересчитывалась при изменении доп. рисков  
**Решение:** База инициализируется при открытии конструктора и пересчитывается только при изменении страховой суммы или скидки

**Файл:** `calculator-variant2-ui.js`

## 📝 Инструкция по обновлению:

1. **Перезагрузите расширение:**
   - Откройте `chrome://extensions/`
   - Найдите "Ингосстрах - Калькулятор страхования"
   - Нажмите кнопку "Обновить" (🔄)

2. **Проверьте горячие клавиши:**
   - Откройте `chrome://extensions/shortcuts`
   - Убедитесь, что установлены:
     - `Alt+Shift+C` для расчета
     - `Alt+Shift+I` для открытия панели

3. **Протестируйте:**
   - Откройте Max или Outlook
   - Попробуйте все функции
   - Проверьте конструктор варианта 2
