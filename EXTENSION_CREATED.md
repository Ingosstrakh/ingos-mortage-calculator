# ✅ Расширение для браузера создано!

## 🎉 Что было создано

Полноценное браузерное расширение (Chrome/Edge) для автоматизации расчетов ипотечного страхования при работе с Max и Outlook.

## 📁 Структура проекта

```
extension/
├── manifest.json              # Манифест расширения (конфигурация)
├── background.js              # Фоновый скрипт (обработка событий)
├── content.js                 # Скрипт для страниц Max/Outlook
├── content.css                # Стили для всплывающих окон
├── sidepanel.html             # HTML боковой панели
├── sidepanel.css              # Стили боковой панели
├── sidepanel.js               # Логика боковой панели
├── setup.ps1                  # Скрипт автоматической установки
├── create-icons.html          # Генератор иконок
├── INSTALL.md                 # Краткая инструкция по установке
└── README.md                  # Полная документация
```

## 🚀 Основные возможности

### 1. Боковая панель (Side Panel)
- Полноценный калькулятор всегда под рукой
- 3 вкладки: Калькулятор, История, Настройки
- Кнопки быстрых действий (извлечь, вставить)
- Автоматическое копирование результата
- Сохранение в историю

### 2. Автоматическое извлечение данных
- Из Max (web.max.ru) - текст сообщений
- Из Outlook - текст писем
- Из выделенного текста на любой странице

### 3. Контекстное меню
- Правый клик → "⚡ Рассчитать страховку"
- Правый клик → "📊 Открыть калькулятор"
- Работает с выделенным текстом

### 4. Горячие клавиши
- `Ctrl+Shift+C` - Рассчитать из выделенного
- `Ctrl+Shift+I` - Открыть боковую панель

### 5. История расчетов
- Сохранение последних 50 расчетов
- Дата, время, источник (Max/Outlook/Калькулятор)
- Клик по записи загружает данные
- Кнопка очистки истории

### 6. Плавающая кнопка
- Появляется на страницах Max и Outlook
- Быстрый доступ к калькулятору
- Можно отключить в настройках

### 7. Всплывающие окна
- Красивое отображение результатов
- Кнопки: Копировать, Вставить, Закрыть
- Автоматическое закрытие
- Toast-уведомления

## 📋 Что нужно сделать для установки

### Шаг 1: Скопировать файлы калькулятора
Запустите PowerShell скрипт:
```powershell
cd extension
.\setup.ps1
```

Или скопируйте вручную все файлы calculator-*.js, config_banks.js, tariffs_*.js, parser.js, installment_calculator.js

### Шаг 2: Создать иконки
Откройте `extension/create-icons.html` в браузере и скачайте иконки

### Шаг 3: Установить в браузер
1. Chrome/Edge → `chrome://extensions/` или `edge://extensions/`
2. Включить "Режим разработчика"
3. "Загрузить распакованное расширение"
4. Выбрать папку `extension`

## 🎯 Как использовать

### Вариант 1: Боковая панель
1. Клик на иконку расширения
2. Ввести данные или нажать "🔍 Извлечь со страницы"
3. Нажать "⚡ Рассчитать"
4. Скопировать или вставить результат

### Вариант 2: Контекстное меню
1. Выделить текст с данными клиента
2. Правый клик → "⚡ Рассчитать страховку"
3. Результат во всплывающем окне

### Вариант 3: Горячие клавиши
1. Выделить текст
2. `Ctrl+Shift+C`
3. Результат во всплывающем окне

## 🔧 Технические детали

### Архитектура
- **Manifest V3** - современный стандарт расширений
- **Service Worker** - фоновая обработка событий
- **Content Scripts** - работа на страницах Max/Outlook
- **Side Panel API** - боковая панель (Chrome 114+)
- **Storage API** - сохранение истории и настроек
- **Context Menus API** - контекстное меню
- **Commands API** - горячие клавиши

### Безопасность
- Все расчеты локально в браузере
- Нет отправки данных на сервер
- История хранится локально
- Минимальные разрешения

### Совместимость
- Chrome 114+ (Side Panel API)
- Microsoft Edge 114+
- Работает на Windows, Mac, Linux

## 📊 Преимущества перед веб-версией

1. **Всегда под рукой** - не нужно переключаться между вкладками
2. **Автоматическое извлечение** - не нужно копировать вручную
3. **Горячие клавиши** - мгновенный доступ
4. **История** - все расчеты сохраняются
5. **Контекстное меню** - расчет в 2 клика
6. **Офлайн работа** - не требует интернета
7. **Интеграция** - работает прямо в Max и Outlook
