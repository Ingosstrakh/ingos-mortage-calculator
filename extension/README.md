# Ингосстрах - Расширение для расчета страхования

Браузерное расширение для автоматизации расчетов ипотечного страхования при работе с Max (web.max.ru) и Outlook.

## 🎯 Возможности

### Основные функции
- **Боковая панель** - полноценный калькулятор всегда под рукой
- **Автоматическое извлечение данных** из Max и Outlook
- **Контекстное меню** - расчет из выделенного текста
- **Горячие клавиши** для быстрого доступа
- **История расчетов** - сохранение последних 50 расчетов
- **Быстрая вставка результата** на страницу или в буфер обмена

### Горячие клавиши
- `Alt+Shift+C` - Рассчитать из выделенного текста
- `Alt+Shift+I` - Открыть боковую панель

### Контекстное меню
- Выделите текст с данными клиента → Правый клик → "⚡ Рассчитать страховку"
- Правый клик на странице → "📊 Открыть калькулятор"

## 📦 Установка

### Способ 1: Установка в режиме разработчика (Chrome/Edge)

1. **Скопируйте файлы калькулятора**
   ```bash
   # Скопируйте все файлы калькулятора в папку extension:
   # - config_banks.js
   # - tariffs_life.js
   # - tariffs_property.js
   # - tariffs_ifl.js
   # - parser.js
   # - installment_calculator.js
   # - calculator-*.js (все модули)
   ```

2. **Откройте Chrome/Edge**
   - Перейдите в `chrome://extensions/` (Chrome) или `edge://extensions/` (Edge)

3. **Включите режим разработчика**
   - Переключите тумблер "Режим разработчика" в правом верхнем углу

4. **Загрузите расширение**
   - Нажмите "Загрузить распакованное расширение"
   - Выберите папку `extension`

5. **Готово!**
   - Расширение установлено и готово к работе
   - Иконка появится на панели инструментов

### Способ 2: Упаковка в .crx (для распространения)

1. В `chrome://extensions/` нажмите "Упаковать расширение"
2. Выберите папку `extension`
3. Получите файл `.crx` для установки на других компьютерах

## 🚀 Использование

### Вариант 1: Боковая панель
1. Кликните на иконку расширения в панели инструментов
2. Откроется боковая панель с калькулятором
3. Введите данные клиента или нажмите "🔍 Извлечь со страницы"
4. Нажмите "⚡ Рассчитать"
5. Результат можно скопировать или вставить на страницу

### Вариант 2: Контекстное меню
1. Выделите текст с данными клиента на странице Max или Outlook
2. Правый клик → "⚡ Рассчитать страховку"
3. Результат появится во всплывающем окне
4. Нажмите "📋 Копировать" или "📝 Вставить"

### Вариант 3: Горячие клавиши
1. Выделите текст с данными клиента
2. Нажмите `Alt+Shift+C`
3. Результат появится во всплывающем окне

### Вариант 4: Плавающая кнопка
1. На страницах Max и Outlook появляется плавающая кнопка 🏛️
2. Кликните на неё для открытия боковой панели

## 🔧 Настройка для Max и Outlook

### Автоматическое извлечение данных

Расширение автоматически определяет тип страницы и извлекает данные:

**Max (web.max.ru)**
- Извлекает текст из активного сообщения в чате
- Поддерживает выделенный текст

**Outlook (outlook.office.com, outlook.live.com)**
- Извлекает текст из открытого письма
- Поддерживает выделенный текст

### Если автоматическое извлечение не работает

1. Выделите текст с данными клиента вручную
2. Используйте контекстное меню или горячую клавишу
3. Или скопируйте текст и вставьте в боковую панель

## 📝 Формат данных

Расширение использует тот же формат, что и веб-калькулятор:

```
ВТБ ост 3000000 муж 15.08.1985 дом кирпич
```

Или:

```
Сбербанк
Остаток: 5000000
Заемщик: жен 20.05.1990
Объект: квартира
Титул
```

## 🐛 Устранение неполадок

### Расширение не работает на странице
- Убедитесь, что вы на странице Max или Outlook
- Обновите страницу после установки расширения
- Проверьте, что расширение включено в `chrome://extensions/`

### Не извлекаются данные со страницы
- Попробуйте выделить текст вручную
- Используйте кнопку "📋 Вставить" в боковой панели
- Скопируйте данные и вставьте в калькулятор

### Ошибка "Модули калькулятора не загружены"
- Убедитесь, что все файлы калькулятора скопированы в папку `extension`
- Проверьте консоль браузера (F12) на наличие ошибок загрузки

### Горячие клавиши не работают
- Проверьте, что горячие клавиши не конфликтуют с другими расширениями
- Перейдите в `chrome://extensions/shortcuts` для настройки

## 🔒 Безопасность и приватность

- Все расчеты выполняются локально в браузере
- Данные не отправляются на внешние серверы
- История хранится локально в браузере
- Расширение не собирает личные данные

## 🎨 Создание иконок

Иконки должны быть в формате PNG с прозрачным фоном:
- `icon16.png` - 16x16 пикселей
- `icon32.png` - 32x32 пикселей
- `icon48.png` - 48x48 пикселей
- `icon128.png` - 128x128 пикселей

Можно использовать любой графический редактор или онлайн-сервис для создания иконок.

## 📞 Поддержка

При возникновении проблем:
1. Проверьте консоль браузера (F12) на наличие ошибок
2. Убедитесь, что все файлы на месте
3. Попробуйте переустановить расширение
4. Обновите браузер до последней версии

## 🔄 Обновление

При обновлении калькулятора:
1. Замените файлы калькулятора в папке `extension`
2. Перейдите в `chrome://extensions/`
3. Нажмите кнопку "Обновить" на карточке расширения
