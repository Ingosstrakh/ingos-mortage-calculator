Поместите сюда иконки расширения:

- icon16.png (16x16 пикселей)
- icon32.png (32x32 пикселей)
- icon48.png (48x48 пикселей)
- icon128.png (128x128 пикселей)

Способы создания иконок:

1. Откройте файл create-icons.html в браузере и скачайте автоматически сгенерированные иконки

2. Используйте онлайн-сервисы:
   - https://www.favicon-generator.org/
   - https://realfavicongenerator.net/

3. Создайте в графическом редакторе (Photoshop, GIMP, Figma)

Требования:
- Формат: PNG
- Прозрачный фон
- Размеры: точно 16x16, 32x32, 48x48, 128x128
- Цветовая схема: синий градиент (#3b82f6 → #2563eb)
- Символ: 🏛️ (здание/банк)
