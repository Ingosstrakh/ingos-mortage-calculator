# ⚡ Быстрый старт - 5 минут

## 1️⃣ Подготовка (2 минуты)

### Скопируйте файлы калькулятора:
```powershell
# Откройте PowerShell в папке extension и выполните:
.\setup.ps1
```

**Или вручную:** скопируйте все файлы `calculator-*.js`, `config_banks.js`, `tariffs_*.js`, `parser.js`, `installment_calculator.js` из корневой папки в папку `extension/`

## 2️⃣ Создайте иконки (1 минута)

1. Откройте файл `create-icons.html` в браузере
2. Нажмите "📥 Скачать все иконки"
3. Сохраните в папку `extension/icons/`

## 3️⃣ Установите расширение (2 минуты)

### Chrome:
1. Откройте `chrome://extensions/`
2. Включите "Режим разработчика" ☑️
3. Нажмите "Загрузить распакованное расширение"
4. Выберите папку `extension`

### Edge:
1. Откройте `edge://extensions/`
2. Включите "Режим разработчика" ☑️
3. Нажмите "Загрузить распакованное расширение"
4. Выберите папку `extension`

## 4️⃣ Готово! Используйте

### Способ 1: Боковая панель
- Клик на иконку расширения 🏛️
- Введите данные
- Нажмите "⚡ Рассчитать"

### Способ 2: Горячая клавиша
- Выделите текст с данными
- Нажмите `Ctrl+Shift+C`
- Результат во всплывающем окне

### Способ 3: Контекстное меню
- Выделите текст
- Правый клик → "⚡ Рассчитать страховку"

## 🎯 Пример использования

1. Откройте Max (web.max.ru) или Outlook
2. Найдите сообщение с данными клиента:
   ```
   ВТБ ост 3000000 муж 15.08.1985 дом кирпич
   ```
3. Выделите текст
4. Нажмите `Ctrl+Shift+C`
5. Результат появится во всплывающем окне
6. Нажмите "📋 Копировать" или "📝 Вставить"

## ❓ Не работает?

### Проблема: Расширение не загружается
**Решение:** Проверьте, что все файлы скопированы и иконки созданы

### Проблема: Ошибка "Модули не загружены"
**Решение:** Запустите `setup.ps1` или скопируйте файлы вручную

### Проблема: Не извлекаются данные со страницы
**Решение:** Выделите текст вручную и используйте горячую клавишу
