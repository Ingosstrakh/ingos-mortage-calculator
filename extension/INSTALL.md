# 🚀 Быстрая установка расширения

## Шаг 1: Подготовка файлов

### Автоматический способ (Windows PowerShell):
```powershell
cd extension
.\setup.ps1
```

### Ручной способ:
Скопируйте следующие файлы из корневой папки в папку `extension/`:
- config_banks.js

- parser.js

## Шаг 2: Создание иконок

### Способ 1: Автоматическая генерация
1. Откройте файл `extension/create-icons.html` в браузере
2. Нажмите "📥 Скачать все иконки"
3. Сохраните файлы в папку `extension/icons/`

### Способ 2: Использование онлайн-сервисов
1. Перейдите на https://www.favicon-generator.org/
2. Загрузите любое изображение (логотип Ингосстрах)
3. Скачайте иконки размеров: 16x16, 32x32, 48x48, 128x128
4. Переименуйте в: icon16.png, icon32.png, icon48.png, icon128.png
5. Поместите в папку `extension/icons/`

## Шаг 3: Установка в браузер

### Chrome:
1. Откройте Chrome
2. Перейдите в `chrome://extensions/`
3. Включите "Режим разработчика" (переключатель в правом верхнем углу)
4. Нажмите "Загрузить распакованное расширение"
5. Выберите папку `extension`
6. Готово! ✅

### Microsoft Edge:
1. Откройте Edge
2. Перейдите в `edge://extensions/`
3. Включите "Режим разработчика" (переключатель слева внизу)
4. Нажмите "Загрузить распакованное расширение"
5. Выберите папку `extension`
6. Готово! ✅

## Шаг 4: Проверка работы

1. Кликните на иконку расширения в панели инструментов
2. Должна открыться боковая панель с калькулятором
3. Введите тестовые данные:
   ```
   ВТБ ост 3000000 муж 15.08.1985 дом кирпич
   ```
4. Нажмите "⚡ Рассчитать"
5. Должен появиться результат расчета

## 🎉 Готово!

Расширение установлено и готово к работе.

### Горячие клавиши:
- `Ctrl+Shift+C` - Рассчитать из выделенного текста
- `Ctrl+Shift+I` - Открыть боковую панель

### Использование:
1. Откройте Max (web.max.ru) или Outlook
2. Выделите текст с данными клиента
3. Правый клик → "⚡ Рассчитать страховку"
4. Или нажмите `Ctrl+Shift+C`

## ❓ Проблемы?

### Расширение не загружается:
- Проверьте, что все файлы скопированы
- Убедитесь, что иконки созданы
- Посмотрите ошибки в консоли расширения

### Не работает на странице:
- Обновите страницу Max/Outlook
- Проверьте, что расширение включено
- Убедитесь, что вы на правильной странице

### Ошибка "Модули не загружены":
- Проверьте, что все файлы calculator-*.js скопированы
- Откройте консоль (F12) и посмотрите ошибки
